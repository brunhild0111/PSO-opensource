The folder 'data/module net' contains test encrypted datasets (x_test.h5, y_test.h5), along with the trained network architecture and parameters (model.json, model.h5).

The activations.py file in 'data_gen' folder contains the generated approximate polynomials.

The utils.py file is primarily used to generate runtime and accuracy metrics, along with auxiliary functions such as batch extraction.

Place the training files in the 'ipynb' folder and the test files in the 'test' folder.
